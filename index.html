🧑🏼
ผิวขาวเหลือง (Type III) เกิดผิวไหม้แดดได้ปานกลาง คล้ำแดดได้ง่ายปานกลาง ไวต่อรังสี UV ปานกลาง
🧑🏽
ผิวสองสี / ผิวน้ำผึ้ง (Type IV) เกิดผิวไหม้แดดได้น้อย คล้ำแดดได้ง่ายมาก มีความต้านทานรังสี UV ปานกลาง-สูง
🧑🏾
ผิวเข้ม / ผิวคล้ำ (Type V-VI) แทบจะไม่ไหม้แดดเลย คล้ำแดดง่ายมาก ต้านทานต่อรังสี UV สูงที่สุด

☀️ UV Index คืออะไร?

ดัชนีรังสี UV มาตรฐาน WHO วัดความเข้มรังสีอัลตราไวโอเลตจากดวงอาทิตย์ ยิ่งสูงยิ่งอันตรายต่อผิวและดวงตา

🛡️ ป้องกัน UV อย่างไร?

UV ≥ 3: ทาครีม SPF 30+ · UV ≥ 6: สวมหมวก แว่ด UV · UV ≥ 8: หลีกเลี่ยงกลางแจ้ง 10:00–16:00

🌫️ PM2.5 อันตรายแค่ไหน?

≤25 µg/m³ ปลอดภัย · 25–50 ปานกลาง · 50–100 เริ่มมีผล · >100 อันตราย ควรสวม N95

🌡️ อุณหภูมิที่เหมาะสม

22–26°C สบายที่สุด · >32°C ควรลดกิจกรรมกลางแจ้ง · >37°C เสี่ยง Heat Stroke
